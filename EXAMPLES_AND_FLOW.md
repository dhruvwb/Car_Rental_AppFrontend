## 📊 Price Calculation Examples

### Example 1: Economy Car Rental
```
Booking Details:
- Car: Maruti Swift (Economy)
- Daily Rate: ₹500
- Days: 2
- Insurance: Basic (₹150/day)
- Add-ons: GPS (₹75/day)

Calculation:
├─ Daily Rate: ₹500 × 2 days = ₹1000
├─ Insurance: ₹150 × 2 days = ₹300
├─ Add-ons: ₹75 × 2 days = ₹150
├─ Subtotal: ₹1450
├─ Tax (18%): ₹261
└─ TOTAL: ₹1711
```

### Example 2: Premium Car Rental
```
Booking Details:
- Car: BMW 3 Series (Premium)
- Daily Rate: ₹3000
- Days: 1
- Insurance: Premium (₹500/day)
- Add-ons: WiFi (₹100), Extra Driver (₹300), GPS (₹75)

Calculation:
├─ Daily Rate: ₹3000 × 1 day = ₹3000
├─ Insurance: ₹500 × 1 day = ₹500
├─ Add-ons: (₹100 + ₹300 + ₹75) × 1 = ₹475
├─ Subtotal: ₹3975
├─ Tax (18%): ₹715.50
└─ TOTAL: ₹4690.50
```

### Example 3: Long Distance Trip
```
Booking Details:
- Distance: 150 KM
- Insurance: None
- Add-ons: None
- Price/KM: ₹1

Calculation:
├─ Distance: 150 km × ₹1/km = ₹150
├─ Insurance: ₹0
├─ Add-ons: ₹0
├─ Subtotal: ₹150
├─ Tax (18%): ₹27
└─ TOTAL: ₹177
```

## 🎯 Navigation Flow

```
Start App
    ↓
[SplashScreen - 2 sec]
    ↓
[Check Auth] ──→ Logged In? ──→ YES ──→ [HomeScreen]
    ↓                                      ↓
   NO                            Select Pickup/Dropoff
    ↓                                      ↓
[LoginScreen]                    [SearchResultsScreen]
    ↓                                      ↓
Login? ──→ YES ──→ [HomeScreen]  Select Car
    ↓                                      ↓
   SKIP                          [CarDetailsScreen]
    ↓                                      ↓
[HomeScreen]                     Select Insurance/Add-ons
                                         ↓
                                [CheckoutScreen]
                                         ↓
                                Confirm Booking
                                         ↓
                            [BookingConfirmationScreen]
```
